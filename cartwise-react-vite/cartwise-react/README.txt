CARTWISE CHAT UPDATE

File gồm 3 phần:
1. index.html - Demo giao diện hoàn chỉnh
2. cartwise-chat.css - CSS cho robot bản cũ, ô chat mới, micro mới
3. cartwise-chat.js - JS mở/đóng chat, kéo thả tự do, resize, gửi tin nhắn, nhập giọng nói

Cách dùng nhanh:
- Mở index.html để xem demo.
- Nếu muốn đưa vào website đang có, copy phần giữa:
  <!-- CARTWISE ASSISTANT START --> ... <!-- CARTWISE ASSISTANT END -->
  vào file HTML chính.
- Import cartwise-chat.css vào <head>.
- Import cartwise-chat.js trước thẻ </body>.

Chỉnh sửa đã làm:
- Đổi hình dáng ô nhập chat theo mẫu mới: thanh bo tròn màu navy, robot mini bên trái, placeholder, micro rõ hơn, nút gửi hình tròn.
- Micro được làm to, nét dày, màu trắng rõ hơn và có trạng thái đang ghi âm.
- Robot dùng style cũ dạng đầu robot đơn giản, không có quai cầm/strap.
- Ô chat có thể kéo thả tự do, không giới hạn trong màn hình.
- Ô chat có thể resize bằng góc dưới bên phải.
